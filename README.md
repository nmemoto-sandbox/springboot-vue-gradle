## Vue.js 
### Project setup
```
cd src/main/js
yarn install
```

### Compiles and hot-reloads for development
```
cd src/main/js
yarn run serve
```

### Compiles and minifies for production
```
cd src/main/js
yarn run build
```

### Lints and fixes files
```
cd src/main/js
yarn run lint
```
